# NCG_jako_solver_regresji_logistycznej-_por-wnanie_z_GD_i_Newtonem
Projekt z optymalizacji ciągłej dotyczący regresji logistycznej. Zaimplementowano i porównano metody Gradient Descent, NCG oraz Newtona-Raphsona na zbiorze breast_cancer, analizując liczbę iteracji, czas działania, wartość funkcji straty i accuracy.
